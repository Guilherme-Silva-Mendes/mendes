# Assets

Place project assets here:

## Required Files

| File | Size | Description |
|------|------|-------------|
| `logo.png` | 200x200 px | Main project logo |
| `logo.svg` | Vector | Scalable logo version |
| `banner.png` | 1280x640 px | GitHub social preview |
| `favicon.ico` | 32x32 px | For documentation site |

## Logo Guidelines

- Use simple, recognizable design
- Works well at small sizes
- Clear on both light and dark backgrounds
- Represents the language's focus on backend/HTTP

## Color Suggestions

Consider using colors that represent:
- Backend/Server: Blues, greens
- Performance: Orange, red accents
- Modern: Gradients or flat design

## Usage

After adding `logo.png`, update the main `README.md`:

```markdown
<p align="center">
  <img src="assets/logo.png" alt="Mendes Logo" width="200">
</p>
```

## Tools for Creating Logos

- [Figma](https://figma.com) - Free, web-based
- [Canva](https://canva.com) - Easy templates
- [Inkscape](https://inkscape.org) - Free, SVG editor
- [GIMP](https://gimp.org) - Free image editor
